# Meta Ad Performance Dashboard (Facebook & Instagram)

## Project Overview
This project involves the development of a comprehensive data analysis ecosystem using **Power BI** to monitor and optimize advertising campaigns across Meta platforms. The primary objective is to transform raw digital marketing data into strategic insights to evaluate ad effectiveness, user behavior, and return on investment.

This is an industry-level project that utilizes a multi-table analysis approach, simulating a real-world, complex data environment.

## Data Architecture & Preparation
The project implements advanced data modeling based on a **Snowflake Schema**. Prior to modeling, a critical **data cleaning and transformation** phase was conducted using Power Query to ensure information integrity. This included removing duplicates, handling null values, and normalizing date and currency formats.

The structure consists of:
- **Fact Table:** Detailed record of ad events.
- **Dimension Tables:** User information, ad catalogs, campaign details, and a customized calendar table.

## Design & Aesthetics
To enhance user experience and visual clarity, the **dashboard aesthetics were designed entirely in Figma**. Custom backgrounds, KPI containers, and an intuitive navigation structure were developed to ensure a professional, modern presentation consistent with Meta's visual identity.

## Metrics & KPIs Implemented
Using **DAX** expressions, the dashboard calculates critical marketing indicators:
- **Reach & Impact:** Tracking impressions and unique views.
- **Engagement:** A combined metric of clicks, shares, and comments.
- **Funnel Efficiency:**
    - **CTR (Click-Through Rate):** Effectiveness of the ad design.
    - **Conversion Rate:** Percentage of users who made a purchase after interacting.
- **Financial Management:** Monitoring total budget and average spend per campaign.

## Key Analysis & Visualizations
- **Consumer Profile:** Engagement analysis broken down by gender and age groups.
- **Format Performance:** Comparison between content types (Video, Carousel, Image, Stories).
- **Temporal Patterns:** Heat maps to identify seasonal trends and peak activity periods.
- **Platform Distribution:** Direct comparison between Facebook and Instagram performance.

## Technologies Used
- **Power BI Desktop:** Data modeling and visualization.
- **Figma:** UI design and aesthetic prototyping.
- **DAX:** Creation of complex measures and calculated columns.
- **Power BI Service:** Cloud publishing and distribution.

## Professional Value
Beyond the technical implementation, this project focuses on **Data Storytelling**, providing a framework to explain findings to stakeholders or during professional interview settings.
